fitur (core) pada aplikasi antrian pelayanan klinik sederhana 
1.Antrian pasien (queue)
2.Riwayat Kunjungan Pasien (Single Linked List)
3.Navigasi Rekam Medis (Double Linked List)
4.Undo/Redo Operasi (Stack)

role pada aplikasi

1. ADMIN KLINIK / Manager
Operasional & Monitoring

Dashboard monitoring real-time antrian
Laporan harian/mingguan/bulanan
Manajemen jadwal dokter dan ruangan
Approve/reject permintaan khusus



tidak bisa hapus data medis atau ubah konfigurasi sistem 


2. PETUGAS PENDAFTARAN / Receptionist
Front Desk Operations

Create: Daftar pasien baru ke antrian
Read: Lihat data pasien dan nomor antrian
Update: Update data demografi pasien
Input data asuransi/BPJS
Cetak nomor antrian
Lihat estimasi waktu tunggu
Pindahkan antrian (reschedule)
Tidak bisa: Akses rekam medis, lihat diagnosa, hapus data


3. DOKTER / Doctor
Medical Services

Read: Akses penuh rekam medis pasien
Create/Update: Input diagnosa, resep, tindakan medis
Lihat riwayat kunjungan lengkap (linked list navigation)
Panggil pasien berikutnya dari antrian pribadi
Rujukan ke dokter spesialis lain
Input catatan medis dan anjuran

Undo/Redo input data (stack feature)
Tidak bisa: Hapus rekam medis lama, ubah data billing


4. APOTEKER / Pharmacist
Medication Management

Read: Lihat resep dari dokter (read-only)
Update: Ubah status resep (diproses/selesai)
Input obat yang diberikan dan jumlah
Cek stok obat
Notifikasi obat habis/menipis
Panggil pasien untuk ambil obat
Hitung total biaya obat yang diberikan
Proses pembayaran pasien untuk obat yang diambil
Cetak struk pembayaran untuk pasien
Menyediakan opsi pembayaran (tunai, kartu kredit/debit, atau metode lainnya)
Tidak bisa: Ubah resep dokter, akses diagnosa lengkap
